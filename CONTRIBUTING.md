# Contributing to Talent Engine AI

Thank you for considering contributing to Talent Engine AI! We appreciate your help and support in making this project better.

## How to Contribute

1. **Fork the repository:**  Click the "Fork" button at the top of the repository page.
2. **Clone your fork:**  Use the following command to clone your fork to your local machine:
   ```bash
   git clone https://github.com/Asad268/talent-engine-ai.git
   ```
3. **Create a new branch:**  Create a new branch for your feature or fix:
   ```bash
   git checkout -b your-feature-branch
   ```
4. **Make changes:**  Make your changes in the code.
5. **Commit your changes:**  Use the following command to stage and commit your changes:
   ```bash
   git commit -m "Add your message here"
   ```
6. **Push to your fork:**  Push your branch to your fork:
   ```bash
   git push origin your-feature-branch
   ```
7. **Create a Pull Request:**  Go to the original repository on GitHub and click the "New Pull Request" button.

## Guidelines
- **Code of Conduct:**  Please verify that your contributions adhere to our Code of Conduct.
- **Follow the existing code style:**  Maintain consistent indentation and style in all code you contribute.
- **Write tests:**  If applicable, write tests for the changes you make to ensure reliability.
- **Documentation:**  Ensure that any changes made to the code base are also documented appropriately.

Thank you for your contribution! We look forward to collaborating with you!
